# Python Course

A beginner-to-advanced Python course covering fundamentals, object-oriented programming, advanced concepts, practice programs, and mini projects. Perfect for students starting their coding journey and aspiring to build real-world Python projects.

---

## Course Structure

- Chapter 1: Introduction & Python Basics  
- Chapter 2: Variables and Data Types  
- Chapter 3: Loops and Conditions  
- Chapter 4: Functions  
- Chapter 5: Lists, Tuples, and Dictionaries  
- Chapter 6: File Handling  
- Chapter 7: Modules and Packages  
- Chapter 8: Object-Oriented Programming (OOP Basics)  
- Chapter 9: Mini Projects  
- Chapter 10: Object-Oriented Project  
- Chapter 11: Inheritance & Advanced OOP Concepts  
- Chapter 12: Advanced Python  
- Stone Paper Scissor: Mini Game Project

---

## Prerequisites

- Python 3.x installed  
- Basic knowledge of programming (optional, but helpful)
